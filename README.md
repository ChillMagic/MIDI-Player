# MIDI-Player
The MIDI Player with Text and Binary File

## Build

### Windows

#### Visual Studio 2017

Open ./VS2017/MIDIPlayer.sln

#### MinGW or Other

Require CMake

```
mkdir build
cd build
cmake ..
make
```

### Linux

Require CMake, Alsa

```
mkdir build
cd build
cmake ..
make
```
